## Feature description

A Flashcard manager that opens in a modal and lists all flashcards available in a table format. 
Filters:
- Filters by tags, content (by fuzzy searching front and back, filename, breadcrumbs). 

Sorting
- Sorts by date reviewed descending
- Sort by due date
- Sort by new 
- Sorts by filename ascending/desc
- Sorts by breadcrumbs

Multiple filters and sorts can be added. 

Each flashcard is selectable. Also the select all in the current filter is availble. 
Flashcards that are selected the option to create or add to a custom deck is availble. 
Creation of custom deck needs a name. 
Custom decks can be viewed on separate tab in Decksview. 
Custom Decks probably need to be managed separately in the database
Flashcards can be in multiple custom decks. 
Reviewing/browsing custom decks is similarly to TagGroup Decks. 
Decks view shows New/Due count for custom decks and the Cog button with same options as Normal decks.
Custom Decks can be backed up and restored.
