### Changes

**Collapsible deck search**
- The filter bar in the decks sidebar is now hidden by default — click the new search icon next to the flashcard manager (Grid) icon to reveal it
- Opening the bar auto-focuses the input so you can start typing immediately
- Press Escape inside the input to clear the query and collapse the bar in one step
- Closing the bar via the toggle preserves your active query, so the same filter snaps back when you reopen it
- Search continues to match against both deck names and tags
- Removed the tag-suggestion dropdown beneath the input in favor of a cleaner, native-Obsidian filter look
