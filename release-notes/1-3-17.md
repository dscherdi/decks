### Changes

**Profile cascade**

- Applying a profile to a parent tag now cascades only to child tags that don't have their own explicit tag mapping
- Applying the default profile to a subtag removes its mapping so it inherits from the parent tag
- Added integration tests for profile cascade behavior

**UI**

- Shortened panel title from "Flashcard Decks" to "Decks"
- Shortened empty state message from "No flashcard decks found" to "No decks found"
