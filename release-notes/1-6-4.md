### Changes

**Backup restore now includes profiles**
- Restoring a backup now also restores deck profiles and tag-to-profile mappings
- Non-default profiles are fully restored from backup; the default profile is preserved if it already exists

**Reset deck progress**
- Added "Reset progress" option to each deck's context menu
- Resets all flashcards in the deck back to new state
- Deletes associated review history and sessions
- Confirmation modal warns about the irreversible action
