## What's new in 1.5.2

### Reverse cards

Add `reverse: true` to the frontmatter of any deck file to automatically generate a reversed version of every card. Each card produces two entries — the original (front → back) and a reversed copy (back → front) — so you practice recall in both directions. Works with all card formats. Review progress is tracked independently for each direction. Removing the flag deletes the reverse cards on the next sync.

### Bug fixes

- Browse mode: pressing Space past the last card no longer increments the index beyond the deck
- Browse mode: clicking Finish now closes the modal instead of showing a notification
- Review mode: modal now closes automatically when the session is complete
- Source file button in card front is no longer keyboard-focusable, preventing accidental activation when pressing Space to advance cards
