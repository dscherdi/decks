### Changes

**FSRS algorithm**

- Upgraded scheduling algorithm from FSRS v4.5 to FSRS v6
- Forgetting curve now uses a trainable decay parameter (w20) for more accurate retention modeling
- Initial difficulty now uses an exponential formula for better calibration across rating levels
- Difficulty updates now apply linear damping, preventing difficulty from getting stuck near the maximum
- Same-day reviews now use a dedicated short-term stability formula (w17–w19) instead of the long-term formula
- Interval calculation updated to match the FSRS v6 specification

**Commands**

- Added "Show release notes" command to view what's new in the current version
- Added "Create test deck" command to generate a sample flashcard file demonstrating table and header-paragraph formats

**New installations**

- A getting-started deck is automatically created on first install, showcasing both supported flashcard formats
