### Decks 2.8.2 — unified deck tree

This release replaces the Files / Tags / Custom tabs in the Decks panel with a single collapsible tree.

#### **One tree, three sections**

The tab switcher is gone. Files, Tags, and Custom are now sections in one list. Files nest into folders by where the deck notes live in your vault; Tags nest by nested-tag path (a tag like `#a/b/c` becomes `a` → `b` → `c`); Custom decks stay a flat list. Pinned decks sit in a "Pinned" block at the top.

#### **Study a whole folder**

Every folder and tag-folder shows combined New and Due counts for everything beneath it, and clicking one starts a review across all the decks it contains. Its options menu also offers browse, cram, and export to Anki for the whole subtree.

#### **Expand/collapse is remembered**

Each branch toggles with the chevron, and the open/closed state is saved and synced across devices. A collapse-all button sits next to the search box, which is now always visible. Typing a filter expands branches so matches show.

Sorting by Deck / New / Due, pinning, the new-card-limit indicator, and the per-row options menu all work as before.
