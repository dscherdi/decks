### Changes

**Decks now speaks 13 languages**

- Full UI translation across English, German, Spanish, French, Italian, Russian, Simplified Chinese, Traditional Chinese, Japanese, Hindi, Albanian, Arabic, and Turkish. Every command, settings field, modal, deck-list label, review button, statistics chart, and notification reads in your chosen language.
- New **Settings → Language → Plugin language** dropdown. Default is **Auto** — the plugin follows Obsidian's UI language and falls back to English for unsupported codes. Pick a specific language to override.
- Changing the language re-renders the settings tab immediately so you can confirm the choice in the new wording; other open views (deck list, review modal, command palette) pick it up on the next plugin reload.
- All translations ship as drafts — pull requests from native speakers welcome.

**Localized "Getting started" note**

- The auto-generated "Getting started" file (created on first install) now matches your active language. Filename, headings, sample table, cloze examples, and the image-occlusion bones are all translated.
- Existing installs are unaffected — the file is only created when missing.

**README in your language**

- The repo README ships with translations for all 12 non-English languages, linked via a language switcher at the top of each file (`README.de.md`, `README.es.md`, …).
