### Decks 2.8.0 — read aloud

This release adds a read-aloud button to review. During a session you can play the current card out loud, with the voice and speed set per profile.

#### **Play button in review**

A speaker button sits with the card's other controls, next to "go to source file", in normal review, browse, and cram. It reads whichever side is showing — the front before you reveal, the answer after. On cloze cards the front is read with the hidden parts spoken as "blank" so the answer isn't given away, and the revealed answer is read in full. Reading stops on its own when you move to the next card, reveal the answer, or close the session; pressing the button again while it's speaking stops it.

#### **Voice and speed are per profile**

Each profile has a new "Read aloud" section in the profiles manager, with a voice picker and a speed slider. The voice list is drawn from the voices installed on your device. Every deck mapped to that profile uses its voice; a profile with no voice set uses the system default.

#### **Works offline, syncs across devices**

Read-aloud uses your device's built-in speech, so it needs no network, account, or key. The chosen voice's language is stored alongside it, so when you sync to a device that doesn't have that exact voice, a voice for the same language is used instead. On platforms without built-in speech the button doesn't appear.

The database migrates automatically on first load after the update.
