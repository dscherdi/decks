## What's new in 1.5.3

### Collapsible breadcrumb navigation

The breadcrumb shown below the session title in the review modal now collapses long paths automatically. Only the last two levels are visible by default — earlier levels appear as `...`. Tap or click any segment to toggle it between its full text and `...`, so you can reveal exactly as much context as you need without the header taking up space. All segments are individually toggleable.
