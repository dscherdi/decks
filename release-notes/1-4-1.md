### Changes

**Deck list modal**

- Added a new modal that displays the full deck list, accessible from the ribbon icon or command palette ("Open review modal")
- Opening a review, browse, config, statistics, or profiles modal from the deck list modal closes it temporarily and reopens it when the child modal is closed
- Navigating to a flashcard's source file from the review modal closes both modals so the file is visible
- Ribbon icon renamed from "Flashcards" to "Decks" and now opens the modal instead of the sidebar panel
