### Changes

**Tab-based review and browse mode**
- Review and browse sessions can now open in a new tab instead of a modal overlay
- Tabs can be dragged to sidebars, bottom panels, or separate windows for flexible positioning
- One review tab at a time — clicking another deck reuses the existing tab
- Tab title updates dynamically to show the deck name and mode (review or browse)

**Settings**
- Added "Review display mode" setting under user interface to switch between modal overlay and new tab

**Stability**
- Stale review tabs from a previous Obsidian session show an "expired" message with a close button instead of crashing
