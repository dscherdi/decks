### Changes

**Custom decks**
- Create custom decks by hand-picking flashcards from any existing deck
- Rename or delete custom decks from the deck list context menu
- Filter-based custom decks: define rules (tag, deck, due status) and the deck updates automatically
- Filter builder UI with live preview of matching cards

**Flashcard manager**
- New manager panel to browse, search, and select flashcards across all decks
- Fuzzy search for quick filtering by card content

**Tag groups include child tags**
- Reviewing a parent tag group (e.g. `#flashcards/math`) now includes all cards from child tags (`#flashcards/math/calculus`, etc.)
- Child tag groups still show only their own cards

**Native UI refresh**
- Deck list panel restyled to match Obsidian's native look and feel
- Header actions use native icon buttons
- Segmented tab control for switching between decks and tag groups
- Deck rows use hover-reveal action menu instead of always-visible gear icon
- Search bar with magnifying glass icon, clear button, and flat dropdown suggestions
- Consistent use of Obsidian CSS variables throughout

**Stability**
- Fixed stale progress bar when switching decks in tab-based review mode
